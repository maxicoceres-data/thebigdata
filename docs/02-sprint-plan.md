## Definición de "Done"

Un issue está **Done** cuando:

- [ ] El PR está mergeado a `main`.
- [ ] Todos los criterios de aceptación del issue están cumplidos.
- [ ] Las queries/dashboards están en la carpeta correcta del repo.
- [ ] La documentación relacionada (si aplica) está actualizada.
- [ ] Comentaste el resultado en el issue antes de cerrarlo.
