# data/

Datos crudos y procesados del proyecto.

## Estructura

- `raw/` — dataset original (.sql, .csv). **No se sube a Git** (ver `.gitignore`).
- `processed/` — datos derivados / limpios (también ignorados por Git).

Ver Issue #2 — `[Data] Carga del dataset classicmodels en MySQL`.
