# powerbi/

Archivos `.pbix` (Power BI Desktop) del proyecto.

## Contenido esperado

- A partir del **Sprint 3** aparecerán acá los primeros archivos.

## Convenciones

- Naming: `<NN>_<descripcion>.pbix` donde `NN` es el número de issue.
- Cada `.pbix` debe tener una **portada** con título, autor, fecha y fuente de datos.
- Paleta corporativa: ver `docs/03-conventions.md`.
